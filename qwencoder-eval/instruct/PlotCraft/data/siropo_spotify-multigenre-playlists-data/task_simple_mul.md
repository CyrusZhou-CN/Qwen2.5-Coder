# Visualization Task - Simple

## Category
Distribution

## Instruction
Create a histogram showing the distribution of energy levels across all music genres in the dataset. Use different colors for each genre and set appropriate transparency (alpha) to handle overlapping distributions. Include a legend, proper axis labels, and a title that clearly indicates what the visualization represents.
