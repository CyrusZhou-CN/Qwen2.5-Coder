# Visualization Task - Simple

## Category
Ranking

## Instruction
Create a horizontal bar chart showing the top 15 strongest currencies against the US Dollar based on their exchange rates. Sort the currencies from highest to lowest exchange rate values, with the currency codes as y-axis labels and exchange rate values on the x-axis. Add appropriate title, axis labels, and format the x-axis to show values with 2 decimal places.
