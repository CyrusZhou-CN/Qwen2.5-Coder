# Visualization Task - Hard

## Category
Groups

## Instruction
Create a comprehensive 3x3 subplot grid analyzing book categories and their characteristics. Each subplot should be a composite visualization: (1) Top-left: Scatter plot with marginal histograms showing price vs quantity colored by star rating, (2) Top-center: Stacked bar chart showing category distribution overlaid with average price line plot, (3) Top-right: Box plot of prices by category with violin plot overlay, (4) Middle-left: Heatmap of category vs star rating counts with annotated values, (5) Middle-center: Radar chart comparing top 5 categories across multiple metrics (avg price, avg quantity, book count), (6) Middle-right: Treemap of categories sized by book count with color intensity representing average price, (7) Bottom-left: Parallel coordinates plot showing relationships between price, quantity, and star rating by category, (8) Bottom-center: Dendrogram clustering categories based on price and quantity characteristics, (9) Bottom-right: Network graph showing category relationships based on similar price ranges and star ratings.
