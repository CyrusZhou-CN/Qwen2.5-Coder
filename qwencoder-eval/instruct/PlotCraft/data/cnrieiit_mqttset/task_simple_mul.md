## Category
Composition

## Instruction
Create a pie chart showing the distribution of different attack types in the MQTT network security dataset. Use the training dataset (train70.csv) to calculate the percentage composition of legitimate traffic versus each type of malicious attack (SlowITe, Bruteforce, Malformed data, Flooding, DoS attack). Display the percentages as labels on each slice and use distinct colors to differentiate between legitimate and malicious traffic categories.
