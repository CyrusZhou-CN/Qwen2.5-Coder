Composition

## Instruction
Create a pie chart showing the composition of total electricity generation by generation type across all provinces and time periods in Canada. Calculate the total megawatt hours for each generation type and display the percentage contribution of each type to the overall electricity generation. Include proper labels, a legend, and use distinct colors for each generation type to clearly show how Canada's electricity generation is composed of different energy sources.
