# Visualization Task - Hard

## Category
Distribution

## Instruction
Create a comprehensive 3x3 subplot grid analyzing the distribution patterns of UAE real estate properties. Each subplot should combine multiple visualization techniques: (1) Top-left: Price distribution histogram with overlaid KDE curve and vertical lines marking quartiles, (2) Top-center: Bathroom count distribution bar chart with overlaid percentage labels, (3) Top-right: Bedroom count distribution pie chart with percentage annotations, (4) Middle-left: Price vs. location violin plot showing price distributions across top 6 most frequent display addresses, (5) Middle-center: Monthly listing addition pattern combining histogram of listing counts per month with overlaid trend line, (6) Middle-right: Price range categorization (budget: <2M, mid-range: 2-4M, luxury: >4M) stacked bar chart by bedroom count, (7) Bottom-left: Box plot of prices by bedroom count with overlaid strip plot showing individual data points, (8) Bottom-center: Ridge plot showing price distributions for different combinations of bedroom-bathroom counts, (9) Bottom-right: Hexbin plot showing the relationship between days since listing addition and price with marginal histograms. Each subplot should include appropriate titles, axis labels, and color schemes to reveal insights about market segmentation, pricing patterns, and temporal trends in the UAE real estate market.
