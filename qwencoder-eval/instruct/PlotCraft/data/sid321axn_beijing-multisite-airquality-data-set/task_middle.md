# Visualization Task - Middle

## Category
Correlation

## Instruction
Create a comprehensive correlation analysis visualization using a 2x2 subplot grid to examine the relationships between air pollutants across Beijing monitoring stations. In the top-left subplot, create a correlation heatmap showing the correlation coefficients between PM2.5, PM10, SO2, NO2, CO, and O3 across all stations combined. In the top-right subplot, create a scatter plot matrix (pair plot) focusing on PM2.5, NO2, and O3 relationships with color-coded points by station. In the bottom-left subplot, generate a bubble plot showing the relationship between PM2.5 (x-axis) and NO2 (y-axis) where bubble sizes represent CO levels and colors represent different stations. In the bottom-right subplot, create a scatter plot with regression lines showing the correlation between PM2.5 and PM10 across all stations, with different colored regression lines for each station to reveal station-specific pollution patterns.
