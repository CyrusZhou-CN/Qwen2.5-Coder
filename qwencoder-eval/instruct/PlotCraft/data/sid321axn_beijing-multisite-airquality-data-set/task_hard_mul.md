# Visualization Task - Hard

## Category
Change

## Instruction
Create a comprehensive 3x2 subplot grid analyzing temporal patterns and trends in Beijing's air quality data across multiple monitoring stations. Each subplot should be a composite visualization combining multiple chart types:

1. Top-left: PM2.5 concentration time series with overlaid 7-day rolling average trend lines for 4 representative stations (Aotizhongxin, Dongsi, Wanliu, Tiantan), including error bands showing daily variance
2. Top-right: Dual-axis plot showing monthly average PM2.5 levels (bar chart) overlaid with temperature trends (line chart) across all 12 stations, with seasonal decomposition indicators
3. Middle-left: Stacked area chart showing the composition of major pollutants (PM2.5, PM10, SO2, NO2, CO, O3) over time for the central Beijing area (average of Dongsi, Tiantan, Wanshouxigong stations)
4. Middle-right: Multi-station comparison using slope charts connecting seasonal averages (Spring, Summer, Fall, Winter) for PM2.5 concentrations, with each line representing a different monitoring station
5. Bottom-left: Calendar heatmap showing daily PM2.5 levels for one representative year (2015) combined with wind direction rose plots for the same period
6. Bottom-right: Time series decomposition plot showing trend, seasonal, and residual components of PM2.5 data for the most polluted station, with autocorrelation analysis overlay

Use consistent color schemes across subplots, include proper legends, and ensure each composite visualization effectively communicates both short-term fluctuations and long-term trends in Beijing's air quality evolution from 2013-2017.
