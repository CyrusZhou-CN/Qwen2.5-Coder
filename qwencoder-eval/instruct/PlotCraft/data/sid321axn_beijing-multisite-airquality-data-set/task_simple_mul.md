# Visualization Task - Simple

## Category
Distribution

## Instruction
Create a histogram showing the distribution of PM2.5 concentrations across all monitoring stations in Beijing. Use data from at least 3 different station files to compare how PM2.5 levels vary by location. Include proper binning (20-30 bins), different colors for each station, and add a legend to identify each station. Set appropriate axis labels and title.

PRSA_Data_Tiantan_20130301-20170228.csv
