# Visualization Task - Middle

## Category
Correlation

## Instruction
Create a comprehensive correlation analysis visualization using a 2x2 subplot grid. In the top-left, display a correlation heatmap showing the relationships between key numerical variables (Monthly_Income, Years_at_Company, Job_Satisfaction, Performance_Rating, Training_Hours_Last_Year, Work_Life_Balance). In the top-right, create a scatter plot with best fit line examining the relationship between Monthly_Income and Years_at_Company, colored by Attrition status. In the bottom-left, show a scatter plot of Job_Satisfaction vs Performance_Rating with point sizes representing Training_Hours_Last_Year. In the bottom-right, create a pair plot focusing on the relationships between Work_Life_Balance, Job_Satisfaction, and Performance_Rating, using different colors for employees with and without attrition.
