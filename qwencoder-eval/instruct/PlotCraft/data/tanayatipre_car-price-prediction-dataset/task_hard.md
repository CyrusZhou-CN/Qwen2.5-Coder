# Visualization Task - Hard

## Category
Groups

## Instruction
Create a comprehensive 3x3 subplot grid analyzing customer segmentation patterns in the credit card dataset. Each subplot should combine multiple visualization techniques: (1) Top-left: Income distribution by employment status with overlaid violin plots and box plots, (2) Top-center: Age vs Employment length scatter plot with density contours and marginal histograms, (3) Top-right: Family composition analysis using stacked bar chart with line overlay showing average income, (4) Middle-left: Education level vs Income correlation heatmap with hierarchical clustering dendrogram, (5) Middle-center: Multi-dimensional radar chart comparing demographic profiles across marital status groups, (6) Middle-right: Asset ownership (car/property) composition using nested pie charts with connecting flow lines, (7) Bottom-left: Employment status network graph showing job title relationships with node sizes representing income levels, (8) Bottom-center: Parallel coordinates plot for all numerical variables colored by risk level with density bands, (9) Bottom-right: Customer lifecycle analysis combining age cohorts with account age using bubble plot overlaid with trend lines.
