# Visualization Task - Middle

## Category
Change

## Instruction
Create a comprehensive 2x2 subplot grid analyzing the temporal evolution of COVID-19 in India's most affected states. Each subplot should be a composite visualization combining multiple chart types:

Top-left: Line chart with area fill showing the progression of total confirmed cases over time for the top 5 states by peak cases, with a secondary y-axis displaying a bar chart of daily new cases for the highest affected state.

Top-right: Stacked area chart showing the composition of cases (Active, Recovered, Deaths) over time for the top 3 most affected states, overlaid with line plots showing the recovery rate percentage for each state.

Bottom-left: Multi-line chart displaying the case fatality rate (Deaths/Total Cases) trends over time for the top 5 states, combined with a scatter plot overlay showing the relationship between total cases and death rate at key time points (every 10th day).

Bottom-right: Dual-axis composite chart with a line plot showing the active cases trend for the top 3 states and a bar chart showing the weekly growth rate of confirmed cases for these same states, with error bars indicating the variability in daily reporting.
