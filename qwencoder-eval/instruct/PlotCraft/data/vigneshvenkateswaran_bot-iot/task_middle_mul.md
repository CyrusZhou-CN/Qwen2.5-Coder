# Visualization Task - Middle

## Category
Composition

## Instruction
Create a comprehensive 2x2 subplot grid analyzing the composition of network attack types in the Bot-IoT dataset. Each subplot should combine multiple visualization techniques:

Top-left: Create a stacked bar chart showing the distribution of attack categories (DoS, DDoS, Reconnaissance, Normal) across different protocols (TCP, UDP, ARP), with each bar segment representing the proportion of each attack type within that protocol.

Top-right: Design a pie chart with an overlaid donut chart - the outer ring shows the overall distribution of attack categories, while the inner donut breaks down the subcategories (TCP, UDP, Service_Scan, etc.) with connecting lines showing the hierarchical relationship.

Bottom-left: Construct a treemap visualization where the main rectangles represent attack categories, subdivided by protocol types, with the size of each rectangle proportional to the number of packets (pkts) and color intensity representing the average bytes per packet.

Bottom-right: Build a horizontal stacked percentage bar chart comparing source IP addresses, where each bar represents 100% and is divided into segments showing the relative composition of different attack categories launched from that IP, with annotations showing the total attack count per IP.
