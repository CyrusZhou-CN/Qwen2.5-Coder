# Visualization Task - Hard

## Category
Groups

## Instruction
Create a comprehensive 3x3 subplot grid analyzing network attack patterns and clustering behaviors in the Bot-IoT dataset. Each subplot should be a composite visualization combining multiple chart types:

Top row (Attack Category Analysis):
- Subplot 1: Stacked bar chart showing attack distribution by protocol (TCP/UDP) with overlaid line plot showing average packet rates
- Subplot 2: Violin plot of bytes distribution by attack category with overlaid box plots showing quartiles
- Subplot 3: Scatter plot of duration vs. packet count colored by attack type with marginal histograms

Middle row (Source Address Clustering):
- Subplot 4: Network graph visualization showing source-destination relationships with node sizes representing traffic volume and edge weights showing connection frequency
- Subplot 5: Parallel coordinates plot showing multi-dimensional attack signatures (bytes, packets, duration, rate) grouped by source address
- Subplot 6: Cluster heatmap of source addresses vs. attack categories with hierarchical clustering dendrogram

Bottom row (Temporal and Behavioral Patterns):
- Subplot 7: Time series plot of attack frequency over time with overlaid moving average trend line, faceted by attack subcategory
- Subplot 8: Radar chart comparing normalized attack characteristics (mean packet size, duration, rate, etc.) across different source addresses
- Subplot 9: Correlation matrix heatmap of network features with overlaid scatter plots in the upper triangle showing feature relationships colored by attack type

The visualization should reveal clustering patterns in botnet behavior, identify distinct attack signatures, and show how different IoT devices exhibit similar malicious behaviors when compromised.
