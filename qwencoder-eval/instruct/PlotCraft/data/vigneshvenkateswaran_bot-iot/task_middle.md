# Visualization Task - Middle

## Category
Composition

## Instruction
Create a 2x2 subplot grid analyzing the composition of network attack types in the Bot-IoT dataset. Each subplot should combine multiple visualization techniques:

Top-left: Create a stacked bar chart showing the distribution of attack categories (Normal, DoS, DDoS, Reconnaissance) across different protocols (tcp, udp, arp), with each bar segment also displaying a line overlay showing the average packet count for each protocol-category combination.

Top-right: Design a pie chart showing the overall composition of attack subcategories (Normal, TCP, UDP, Service_Scan), with each slice annotated with both percentage and absolute count, plus add a secondary donut chart in the center showing the split between malicious (attack=1) vs benign (attack=0) traffic.

Bottom-left: Construct a horizontal stacked bar chart displaying the composition of source IP addresses contributing to each attack category, with bars colored by attack type and overlaid with scatter points indicating the average bytes transferred per source IP.

Bottom-right: Build a treemap visualization showing the hierarchical composition of attack types (category -> subcategory -> protocol), with rectangle sizes proportional to the number of records and colors representing the average duration of attacks, supplemented by text annotations showing the percentage contribution of each leaf node.
