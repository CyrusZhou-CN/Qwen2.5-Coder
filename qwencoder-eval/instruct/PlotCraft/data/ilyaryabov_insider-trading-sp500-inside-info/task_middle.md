# Visualization Task - Middle

## Category
Deviation

## Instruction
Create a comprehensive deviation analysis visualization showing how insider trading transaction values deviate from company-specific baselines across different S&P 500 companies. Design a 2x2 subplot grid where each subplot combines multiple visualization techniques: (1) Top-left: A diverging bar chart overlaid with error bars showing mean transaction values per company compared to the overall S&P 500 average, with companies sorted by deviation magnitude; (2) Top-right: A dumbbell plot comparing average "Buy" vs "Sale" transaction values for each company, highlighting the gap between these transaction types; (3) Bottom-left: A radar chart displaying the top 10 companies by transaction volume across multiple deviation metrics (value deviation, frequency deviation, and timing concentration); (4) Bottom-right: An area chart showing cumulative deviation from the rolling 30-day average transaction value over time, with separate areas for positive and negative deviations. Use color coding to distinguish between different relationship types (CEO, Director, etc.) and ensure each subplot includes appropriate reference lines for baseline comparisons.
