# Visualization Task - Middle

## Category
Correlation

## Instruction
Create a comprehensive correlation analysis visualization showing the relationships between EEG brainwave patterns across different electrode positions and motor imagery states. Design a 2x2 subplot grid where: (1) Top-left: A correlation heatmap showing Pearson correlations between all Alpha band measurements (Alpha_TP9, Alpha_AF7, Alpha_AF8, Alpha_TP10) across all subjects, with a diverging color scheme and correlation coefficients annotated; (2) Top-right: A scatter plot matrix (pair plot) focusing on Beta band measurements, with different colors representing different concentration levels (binned into Low: 0-33, Medium: 34-66, High: 67-100), including regression lines for each concentration group; (3) Bottom-left: A bubble plot showing the relationship between Delta_TP9 (x-axis) and Theta_TP10 (y-axis), where bubble sizes represent Gamma band average across all electrodes, and colors indicate Mellow state levels; (4) Bottom-right: A correlation network plot displaying the strongest correlations (|r| > 0.3) between all brainwave bands across all electrode positions, with edge thickness proportional to correlation strength and node colors representing electrode positions.
