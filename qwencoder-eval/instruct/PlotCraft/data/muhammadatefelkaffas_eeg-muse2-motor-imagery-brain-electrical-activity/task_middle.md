# Visualization Task - Middle

## Category
Correlation

## Instruction
Create a composite visualization analyzing the relationships between EEG brainwave patterns and cognitive states in motor imagery tasks. Design a 2x2 subplot grid where: (1) Top-left: Create a correlation heatmap showing the relationships between all five brainwave bands (Delta, Theta, Alpha, Beta, Gamma) across all four electrode positions (TP9, AF7, AF8, TP10), (2) Top-right: Generate a scatter plot matrix (pair plot) focusing on Alpha and Beta bands across the four electrodes, with points colored by concentration levels, (3) Bottom-left: Create a bubble plot showing the relationship between Mellow and Concentration scores, where bubble sizes represent the average Alpha wave activity across all electrodes, and (4) Bottom-right: Design a network-style correlation plot displaying only strong correlations (|r| > 0.5) between brainwave features, with edge thickness representing correlation strength. This analysis should reveal how different brainwave patterns correlate with each other and with measured cognitive states during motor imagery experiments.
