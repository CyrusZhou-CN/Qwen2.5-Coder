# Visualization Task - Hard

## Category
Groups

## Instruction
Create a comprehensive 3x3 subplot grid analyzing EEG brainwave patterns across different motor imagery states and electrode positions. Each subplot should be a composite visualization combining multiple chart types:

Row 1: Electrode-based clustering analysis
- Subplot 1: Scatter plot with density contours showing Alpha vs Beta power relationships across all four electrodes (TP9, AF7, AF8, TP10), with different colors for each electrode and overlaid KDE contours
- Subplot 2: Hierarchical clustering dendrogram of electrode similarities based on all five frequency bands (Delta, Theta, Alpha, Beta, Gamma), with a heatmap showing correlation coefficients between electrodes
- Subplot 3: Parallel coordinates plot displaying normalized frequency band profiles for each electrode, with violin plots showing distribution density at each coordinate

Row 2: Frequency band interaction networks
- Subplot 4: Network graph showing correlations between frequency bands across electrodes as nodes and edges, overlaid with a correlation matrix heatmap in the background
- Subplot 5: Radar chart comparing average frequency band patterns for different concentration levels (binned into low/medium/high), with error bars and filled areas showing confidence intervals
- Subplot 6: Bubble plot showing Theta vs Gamma relationships with bubble sizes representing Delta power, colored by electrode position, with marginal box plots

Row 3: Temporal and battery-related groupings
- Subplot 7: Time series cluster analysis showing grouped patterns of Alpha waves over timestamp sequences, with different line styles for each identified cluster and shaded confidence bands
- Subplot 8: Stacked area chart showing the composition of all frequency bands over battery levels, overlaid with a line plot showing concentration trends
- Subplot 9: Multi-dimensional scatter plot matrix (pairplot style) showing relationships between Mellow, Concentration, and dominant frequency band ratios, with regression lines and correlation coefficients displayed
