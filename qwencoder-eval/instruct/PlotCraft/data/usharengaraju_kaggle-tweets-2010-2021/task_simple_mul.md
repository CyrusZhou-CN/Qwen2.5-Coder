# Visualization Task - Simple

## Category
Change

## Instruction
Create a line chart showing the evolution of tweet engagement over time from 2010 to 2021. Calculate the average engagement rate (likes_count + retweets_count + replies_count) per tweet for each year and plot it as a single line chart to reveal how Kaggle-related tweet engagement has changed over the decade.
