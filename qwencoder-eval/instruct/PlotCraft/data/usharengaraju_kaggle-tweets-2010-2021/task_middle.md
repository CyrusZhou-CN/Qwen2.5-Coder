# Visualization Task - Middle

## Category
Change

## Instruction
Create a comprehensive temporal analysis of Kaggle's social media engagement evolution from 2010-2021. Design a 2x2 subplot grid where each subplot combines multiple visualization techniques: (1) Top-left: Line chart showing tweet volume per year overlaid with a bar chart showing average engagement metrics (likes + retweets) per tweet by year, (2) Top-right: Stacked area chart displaying the proportion of different languages used in tweets over time with a secondary y-axis line showing total tweet diversity, (3) Bottom-left: Time series plot of monthly tweet patterns with seasonal decomposition showing trend, seasonal, and residual components, (4) Bottom-right: Dual-axis visualization combining a line chart of yearly hashtag usage frequency with a scatter plot showing the relationship between tweet length and engagement over time.
