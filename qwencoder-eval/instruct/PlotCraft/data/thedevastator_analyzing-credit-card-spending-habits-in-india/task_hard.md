# Visualization Task - Hard

## Category
Change

## Instruction
Create a comprehensive 3x3 subplot grid analyzing credit card spending patterns over time across different dimensions. Each subplot should combine multiple visualization techniques: (1,1) Monthly spending trends with overlaid moving average line and confidence bands, (1,2) Seasonal spending patterns by card type with stacked area chart plus trend lines, (1,3) Daily spending distribution with histogram and KDE overlay, (2,1) City-wise spending evolution over time with multiple line plots and filled areas, (2,2) Card type performance trends with bar charts and cumulative spending line overlay, (2,3) Weekly spending patterns with box plots and violin plot overlay, (3,1) Quarterly spending analysis with grouped bars and trend line, (3,2) Month-over-month growth rates with line chart and error bars, (3,3) Time series decomposition showing trend, seasonal, and residual components with multiple overlaid plots.

## Files
Credit card transactions - India - Simple.csv
